# TITASBET - HTML CSS JS

Este site é 100% estático. Não precisa de Node, npm, Vite nem build.

## Como publicar no GitHub Pages

1. Cria um repositório no GitHub.
2. Faz upload destes ficheiros todos:
   - index.html
   - style.css
   - script.js
   - pasta assets
3. Vai a Settings > Pages.
4. Em "Build and deployment", escolhe:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
5. Guarda.
6. Espera 1-2 minutos.

O site vai abrir num link tipo:

https://teunome.github.io/nome-do-repo/
