# TITASBET - HTML CSS JS

Site 100% estático com saldo fake e aposta única demo.

## Publicar no GitHub Pages

1. Cria ou abre o teu repositório no GitHub.
2. Apaga os ficheiros antigos.
3. Faz upload destes ficheiros:
   - index.html
   - style.css
   - script.js
   - pasta assets
4. Vai a Settings > Pages.
5. Escolhe:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
6. Guarda e espera 1-2 minutos.

Não precisa de terminal, npm, Node ou Vite.
